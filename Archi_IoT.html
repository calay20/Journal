The following are the Architecture of Internet of Things(IoT):

  1. Sensing Layer
    • The main purpose of the sensing layer is to identify any phenomena in the devices’ peripheral and obtain data from the real world.
    • This layer consists of several sensors. Using multiple sensors for applications is one of the primary features of IoT devices.
    • Sensors in IoT devices can be classified into three broad categories such as; (1) Motion sensors, measure the change in motion as well as the orientation of the devices. (2) Environmental Sensors, are embedded in IoT devices to sense the change in environmental parameters in the device’s peripheral and the (3) Position sensors which deals with the physical position and location of the device.
  2. Network Layer
    • The network layer acts as a communication channel to transfer data, collected in the sensing layer, to other connected devices.
  3. Data Processing Layer
    • The data processing layer consists of the main data processing unit of IoT devices. The data processing layer takes data collected in the sensing layer and analyses the data to make decisions based on the result.
  4. Application Layer
    • The application layer implements and presents the results of the data processing layer to accomplish disparate applications of IoT devices. The application layer is a user-centric layer that executes various tasks for the users.
